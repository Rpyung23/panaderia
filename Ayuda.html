Introducción

El presente proyecto es una aplicación web que permite gestionar el almacenamiento de datos de: productos, proveedores, y las ventas de una Panadería de zapatería. La página web cuenta con: inicio de sesión, gestión de productos: añadir nuevo producto, buscar un producto, modificar datos del producto, y eliminar datos del producto, gestión de proveedores: añadir nuevo proveedor, buscar proveedor, modificar datos del proveedor y eliminar datos del proveedor, gestión de ventas: añadir nueva venta, imprimir factura, y reportes
